迷因怎麼貼到 LINE / IG / Threads / FB Messenger：每平台複製方式

By TWmeme · 更新於 2026-05-12

30 秒快答：LINE 和 Threads 支援自動展開連結預覽、所以「複製連結 → 直接貼」最快。IG 訊息對連結預覽不友善、建議「下載原圖 → 上傳圖片」。FB Messenger 兩種都行、但連結會顯示完整 URL 不夠美觀、推薦下載再傳。

同樣是貼迷因圖、不同平台行為差很多。下面四個平台的最佳實踐、加上 TWmeme 對應的按鈕用法。

LINE：複製連結最快

LINE 對連結預覽支援很好——貼一個圖片 URL（jpg / png / webp）進對話、LINE 會自動展開成完整預覽圖、點開可以看大圖。優點：

  • 過程最快（不用下載再上傳）
  • 不佔對方手機相簿
  • 群組訊息保留更久（圖片過期 LINE 還能看連結原圖）

缺點：

  • 連結文字會顯示在預覽下面、看起來有點「貼網址」
  • 如果原圖床掛了預覽會 broken（TWmeme 用 R2 緩存所以這個風險低）

TWmeme 操作：詳細頁 →「複製連結」→ 切到 LINE → 長按輸入框「貼上」→ 送出。

IG 訊息（Instagram DM）：下載原圖比較好

Instagram 對訊息裡的連結預覽很冷淡——大部分情況下只顯示一段文字 URL、不展開預覽。所以 IG 訊息的最佳實踐是下載原圖 → 從相簿上傳：

  • 對方看到的是真正的圖片訊息、不是連結卡片
  • 可以用 IG 的圖片回應功能（emoji 反應）
  • 送出 IG 限時動態的素材也走這條路徑

TWmeme 操作：詳細頁 →「下載原圖」（手機版會存到相簿、桌面版到下載資料夾）→ 切到 IG → 訊息附件 → 選相片 → 送出。

Threads：複製連結很好用

Threads（Meta 的 Twitter-like）跟 IG 不一樣、對連結預覽友善。貼圖片 URL 會展開預覽圖、留言看起來乾淨。Threads 的演算法似乎也會給有圖的貼文比較多曝光、所以做內容創作的人會偏好「複製連結 → 直接貼」這個路徑。

TWmeme 操作：詳細頁 →「複製連結」→ 切到 Threads → 新貼文 → 貼上 → 等預覽出現 → 送出。

FB Messenger：看情境

Messenger 兩種方式都能用、但體驗稍有不同：

  • 貼連結：會展開預覽、但同時顯示完整 URL 字串（不美觀）
  • 下載再上傳：純圖片訊息、最乾淨；可以用 Messenger 的反應功能

推薦下載再傳。FB 粉專小編做封面圖、IG 限動素材、貼到 FB 一般動態——都建議走下載路徑。

桌面 vs 手機

差異主要在「下載原圖」這個動作：

  • 桌面版瀏覽器：「下載原圖」會把檔案存到「下載」資料夾、檔名是 TWmeme 自動產的「標題.jpg」格式（特殊符號會被替掉）。然後拖曳到聊天視窗或用平台的「附件」功能。
  • 手機版瀏覽器：「下載原圖」會打開圖片在新分頁、長按可以「儲存圖片」到相簿。iOS Safari 和 Chrome 都支援。Android 直接「下載圖片」存到下載資料夾。

手機版有個快路徑：直接長按 TWmeme 詳細頁上的圖片、瀏覽器會跳出「儲存圖片」「複製圖片」選單、不需要點按鈕。複製圖片之後可以直接貼到聊天視窗（比連結 URL 更直觀）。

分享按鈕在做什麼

TWmeme 詳細頁第三個按鈕「分享」其實分享的是那一頁的 TWmeme URL（不是圖片 URL）、讓對方點進來看你找到的這張迷因。手機版會呼叫系統原生分享面板（可以分享到 LINE、IG、Threads、AirDrop 等任何 app）；桌面版降級成複製目前頁面 URL。

用「分享」傳給朋友、對方看到的是 TWmeme 頁面、有完整的下載 / 複製選項；用「複製連結」傳的是圖片 URL、對方看到的是純預覽圖。看情境選。

常見問題

為什麼貼到 IG 訊息只有文字、沒展開圖？

IG 訊息對連結預覽支援不好、大部分情況下不展開。這是 IG 平台限制不是 TWmeme 的問題——同樣的 URL 貼到 LINE 或 Threads 會展開。在 IG 訊息建議改用「下載原圖 → 上傳圖片」流程、收件人看到的是真正的圖片訊息。

複製連結的 URL 是 TWmeme 還是原圖床？

是 Cloudflare R2 的緩存 URL（pub-xxxxx.r2.dev/memes/.jpg 格式）、不是 imgur 或 meee.com.tw 等原始圖床。這樣有兩個好處：原圖床掛了也不影響、不會給原始上傳者帶來流量壓力。原始 PTT 文章連結會在詳細頁「看原文」按鈕保留、想追溯來源隨時可以。

下載的檔名為什麼有奇怪符號被替掉？

瀏覽器的 download 機制對檔名有限制——不能有 `/ \ ? % * : | " < >` 這些字元（會被作業系統當成路徑符號）。TWmeme 在下載時把這些字元清掉、用 PTT 標題的可顯示部分當檔名。如果原標題只有特殊符號（罕見），會 fallback 成「meme.jpg」。
